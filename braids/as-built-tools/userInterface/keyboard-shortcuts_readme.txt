
Keyboard Shortcuts

Layer Navigation
Ctrl+Numpad5	switch to windowAnchor layer
Ctrl+Numpad4	switch to wallUnbroken layer
Ctrl+Numpad3	switch to wallThicknessHelper layer
Ctrl+Numpad2	switch to doorGuideNonPivotingSide layer
Ctrl+Numpad1	switch to doorAnchor layer


Constraints
Ctrl+D			show all dimensions
Ctrl+Shift+D	hide all dimensions
Ctrl+F			show all constraintss
Ctrl+Shift+F	hide all constraintss

Dimensioning
Ctrl+Shift+A	adds a dcaligned dimension with minimal user interaction

Zooming
Ctrl+W			zoom all
Ctrl+K			zoom in
Ctrl+M			zoom out
